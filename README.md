## Commands

**Update both Rust and Cargo**

```bash
→ rustup update
```

**Check the current Cargo version**

```bash
→ cargo --version
```

**Compile and make an executable**

```bash
→ cargo build
```

**With optimization**

```bash
→ cargo build --release
```

**Make sure the codes compile**

```bash
→ cargo check
```

**Run the executable**

```bash
→ cargo run
```

**Build documentation of dependencies and open in browser**

```bash
→ cargo doc --open
```
